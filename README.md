# tkinter-calculator
A simple and user-friendly calculator application built with Python and Tkinter, featuring a graphical user interface and basic arithmetic operations.

# Simple GUI Calculator 🧮

A lightweight and user-friendly Graphical User Interface (GUI) calculator built using Python and the `tkinter` library. This project is perfect for performing quick, basic mathematical computations.

## 🌟 Features

* **Basic Arithmetic:** Addition (`+`), Subtraction (`-`), Multiplication (`*`), and Division (`/`).
* **Advanced Operations:** * Power/Square (`x²`)
    * Percentage (`%`)
    * Pi constant (`pi`)
* **Clear Display:** Easy-to-read input and output field.
* **Error Handling:** Displays `ERROR` for invalid operations (e.g., dividing by zero).
* **Clean Interface:** Color-coded buttons for numbers, operations, and clear functions.

## 🚀 How to Run

### Prerequisites
Make sure you have Python installed on your system. `tkinter` is included in the standard Python library, so no extra installation is required.

### Execution
1. Clone this repository:
   git clone [https://github.com/parniiyan/tkinter-calculator.git](https://github.com/YOUR-USERNAME/YOUR-REPOSITORY.git)

2. Navigate to the project directory:
   cd YOUR-REPOSITORY

3. Run the Python script:
   python pro_gui_calculator.py


*(Note: Depending on your system, you might need to use `python3` instead of `python`)*

## 🛠️ Technologies Used

* **Language:** Python 3
* **Library:** Tkinter (for the GUI)
